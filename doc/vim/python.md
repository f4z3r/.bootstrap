## Coc Extension

Install the `coc` extension.

```viml
CocInstall coc-python
```

## Tools

Use `pipx` to install python executable tools.

```sh
$ pip3 install --user pipx
```

`pipx` can then be used to install tools.

```sh
$ pipx install <tool>
```
